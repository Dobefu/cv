## Getting Started

- Copy `.env.example` to `.env`

  ```bash
  cp .env.example .env
  ```

- Fill in all the fields in the `.env` file

- Install the dependencies

  ```bash
  pnpm i
  ```

- Run the development server

  ```bash
  pnpm dev
  ```

Open [http://localhost:3000](http://localhost:3000) in your browser of choice to see the result.
